# Projeto de Clusters de Shopping

Projeto desenvolvido originalmente em 08/2025 para o módulo 33 do curso de ciência de dados da EBAC. Por meio de uma base de dados contendo 4 características de frequentadores de um shopping (gênero, idade, salário anual e score de gasto), desenvolvi um projeto de ciência e análise de dados com objetivo de separar esses clientes em grupos (clusters), utilizando o modelo de aprendizado de máquina K-means para analisar os padrões das características presentes e identificar os clientes presentes em cada grupo e permitindo que estratégicas de marleting direcionadas a esses grupos possam ser feitas.

O passo a passo do projeto é constituído pela análise e tratamento dos dados, pré-processamento das informações, modelagem e avaliação dos resultados, contando com comentários e explicações de cada etapa, insights e conclusões que foram possíveis de serem retirados e gráficos para visualização prática dos fatores, possibilitando um storytelling prático. Foi desenvolvido utilizando a linguagem Python e bibliotecas essenciais para a área de dados: Pandas, Matplotlib, Seaborn, Plotly e Scikit-learn, no Jupyter Notebook.

Para o funcionamento eficaz do código, é necessário ter instalado todos os pacotes e bibliotecas presentes no arquivo 'requirements.txt'.

O aluno e cientista de dados Guilherme Zorzetto de Sá foi o principal colaborador desse projeto, proposto e feito em parceria com a EBAC e seus tutores.
